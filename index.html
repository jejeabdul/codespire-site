Code + Inspire

Code Inspiration adalah tool untuk membuat dan menyiapkan konten edukasi pemrograman berformat video pendek, lalu mengunggahnya sebagai draft ke akun TikTok pemilik untuk ditinjau dan dipublikasikan secara manual.

Cara kerja

Aplikasi membuat aset video edukasi, lalu menggunakan TikTok Login Kit dan Content Posting API (scope video.upload) untuk mengirim video sebagai draft ke kotak masuk (inbox) akun TikTok yang telah memberi otorisasi. Publikasi akhir selalu dilakukan secara manual oleh pemilik akun di dalam aplikasi TikTok.

Hubungkan akun TikTok

Beri otorisasi agar Code Inspiration dapat mengirim video sebagai draft ke Inbox TikTok kamu. Izin yang diminta:

user.info.basic video.upload

Setelah login & memberi izin, kamu diarahkan kembali ke callback.html. Publikasi akhir selalu kamu lakukan manual di app TikTok.
